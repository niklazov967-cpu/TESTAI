# Чекпоинты проекта

## Checkpoint 0 (v0.0.0)

**Дата создания:** 2025-01-23  
**Git Tag:** `v0.0.0`  
**Commit:** `fe610d8`

### Описание

Начальная версия проекта с полной поддержкой титана (Ti) в химическом составе.

### Основные функции

✅ **Трёхэтапная архитектура обработки:**
- Stage 1: Tavily API - Поиск данных о сталях
- Stage 2: DeepSeek API - Обработка и поиск аналогов
- Stage 3: OpenAI API - Валидация и фактчекинг

✅ **Химический состав:**
- Полный набор элементов: C, Mn, Si, Cr, Ni, Mo, V, **Ti**, P, S, Fe
- **Титан (Ti) обязателен** для всех сталей
- Автоматическое определение титаностабилизированных сталей

✅ **Функциональность:**
- Поиск аналогов сталей для трёх стран (США, Россия, Китай)
- Расчет углеродного эквивалента (CE)
- Классификация сталей
- Оценка свариваемости
- Валидация результатов
- Кэширование результатов

✅ **Интерфейс:**
- Веб-интерфейс (HTML/CSS/JS)
- REST API endpoints
- Детальное отображение результатов

### Известные особенности

- Титан (Ti) всегда присутствует в химическом составе (0 если отсутствует)
- Поддержка титаностабилизированных сталей (например, AISI 321, 08Х18Н10Т)
- Кэширование включено по умолчанию
